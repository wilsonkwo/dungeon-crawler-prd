## ✨ Features
- **Turn-based combat** — Strategic positioning matters
- **Stat-based loot** — 45+ weapons, armor, shields, necklaces
- **Class system** — Fighter, Thief, Mage with unique stats/gear
- **Merchant system** — Buy keys, identify scrolls, upgrade kits
- **Trait system** — Vampirism, Fortune, Guard, Resilience effects
- **Leaderboard** — Top 10 scores with LocalStorage
- **Mobile-friendly** — Tap-to-move, responsive design
- **Visual polish** — Gradient UI, sprite effects, smooth camera, particles

## 🎯 Controls
- **Desktop:** Arrow keys / WASD, F=Shoot, G=Magic, P=Potion
- **Mobile:** Tap near hero to move, tap enemy to attack
- **Both:** Click action buttons for special abilities
